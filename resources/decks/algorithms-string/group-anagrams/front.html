Дан массив строк, нужно сгруппировать в нем анаграммы. Важно, что это нужно сделать максимально эффективно по памяти и времени. Слово X является анаграммой слова Y, если оно может быть получено из другого перестановкой букв. 

    Пример:

    [
        "cot", "toc", "el", "elm", "s", "lem"
    ] 
    ->
    [
        ["cot", "toc"],
        ["lem",  "elm"],
        ["el"],
        ["s"]
    ]
